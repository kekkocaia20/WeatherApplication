Il sito è una Weather App 🌦️ che permette di consultare le previsioni meteo in tempo reale.
Mi sono appoggiato a un’API esterna 🌐 per ottenere dati aggiornati per qualsiasi città.

Per lo sviluppo ho utilizzato:

- Python 🐍 e Flask ⚡ per la logica server-side e la gestione delle richieste;

- HTML 📄 e CSS 🎨 per la struttura e lo stile della pagina;

- Bootstrap 📱 per rendere il design responsive e coerente su desktop e mobile.

Il risultato è un’app semplice, moderna e funzionale, con layout chiaro e interattivo.
